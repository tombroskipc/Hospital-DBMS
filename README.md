# Hospital DBMS written in Flask and MySQL

# FLASK SETTINGS:

## Windows PowerShell

```powershell
$env:FLASK_APP="__init__"
$env:FLASK_ENV="development"
```

## Windows CMD

```
set FLASK_APP=__init__
set FLASK_ENV=development
```

## Linux Bash

```bash
export FLASK_APP=__init__
export FLASK_ENV=development
```

# Run flask:
```
flask run
```
